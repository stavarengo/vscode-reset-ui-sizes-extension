# Reset Sizes Extension

This extension provides a command to reset all sizes in your application to their default values. It is designed to help developers quickly revert any size changes made during development.

## Features

- **Reset All Sizes**: A command that resets the sizes of various elements in the application.
- **Customizable Options**: Allows for customization of reset options through the extension's settings.

## Installation

1. Clone the repository or download the ZIP file.
2. Open the project in your code editor.
3. Run `npm install` to install the necessary dependencies.
4. Open the command palette and type `Reload Window` to activate the extension.

## Usage

To use the extension, open the command palette (usually `Ctrl+Shift+P` or `Cmd+Shift+P` on macOS) and type `Reset All Sizes`. Select the command to execute it.
